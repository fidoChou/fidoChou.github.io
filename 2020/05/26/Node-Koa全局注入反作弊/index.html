中间件

为什么叫中间件？因为出于 Http 的 Request 和 Response 中间，用来实现某种功能。

我的老板提出这个问题时，我几乎立刻抖了个机灵：写个中间件呗？（因为我室友就是专业写中间件的，hahaha）

老板：嗯？Koa 基于洋葱模型，那能拿到 http 状态吗？

抖机灵：从 ctx 取呗？

老板：那 C 端怎么做呢？

抖机灵：从 fetch 请求直接 reject 出来！
